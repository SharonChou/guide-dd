## 智能日期识别规则示例

你可以在“快速添加”中直接输入日期文本，滴答清单会自动识别并为你创建日期和提醒。

### 日期和时间

| 你输入的文本 | 滴答清单会识别成 |
| --- | :---: |
| 今天 | 今天 |
| 明天 | 明天 |
| \(今天 **或** 已过期\) **且** 高优先级 | 显示“今天”或“已过期”里高优先级的任务 |
| \(\#电脑旁 **且** \#工作\) | 显示同时包含标签“\#电脑旁”和“\#工作”的任务 |
| 收集箱 **且** 没有日期 | 显示收集箱清单里的没有日期的任务 |
| \(工作 **或** 团队\) **且** 高优先级 | 显示“工作”或“团队”清单里高优先级的任务 |
